## 📌 Project Overview

Agricultural production forecasting is essential for **food security, planning, and policy-making**.

This project processes agricultural datasets, transforms them into a structured format, and builds a **Machine Learning model** to predict crop production based on key factors.

The dashboard allows users to explore:

• Crop production prediction  
• Regional crop analysis  
• Yield vs Area relationship  
• Agricultural trend insights  

---

## 🚀 Features

✔ Data Cleaning and Preprocessing
